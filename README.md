# Uruchomienie projektu

## Instalacja zależności

W folderze głównym projektu uruchom:
npm install

## Instalacja i kompilacja modułu Rust

Przejdź do folderu rust_module:
cd rust_module
npm install
npm run build

## Uruchomienie aplikacji

Wróć do folderu głównego:
cd ..
npm run dev

Aplikacja uruchomi się w trybie deweloperskim.
